## Usage:

There are 5 log levels:

1. ```ts
   log.trace(...objs);
   ```

   A debug message. This can be configured to not be displayed if a verbose flag is not set, and by default prints as a lower contrast color.

2. ```ts
   log.info(...objs);
   ```

   An informational message. Prints to stdout.

3. ```ts
   log.warn(...objs);
   ```

   A warning message. Prints to stderr, and by default prints as yellow.

4. ```ts
   log.error(...objs);
   ```

   An error message. Prints to stderr, and by default prints as red.

5. ```ts
   log.fatal(...objs);
   ```

   A fatal error message. Prints to stderr, and by default prints as bold red and exits the program (configurable).
